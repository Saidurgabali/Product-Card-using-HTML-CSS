# Task-2: Styling a Product Card

## Description
This project demonstrates the creation of a simple and responsive product card using **HTML** and **CSS**. The card includes a product image, product name, price, and an **“Add to Cart”** button. CSS styling is applied to ensure proper alignment and a clean, professional appearance, along with hover effects for better user interaction.

## Features
- Product image display  
- Product name and price  
- Add to Cart button with hover effect  
- Clean and well-aligned card layout  

## Technologies Used
- HTML5  
- CSS3  

## Purpose
This task helps in understanding basic UI design concepts, CSS styling, alignment techniques, and interactive hover effects for web components.
